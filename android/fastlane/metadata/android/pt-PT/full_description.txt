Este é um gerador de palavras do Major System. O Major System é um sistema de memória de números para palavras para ajudar você a lembrar números, datas, etc. Cada um dos 10 dígitos do sistema é representado por um conjunto de consoantes com sons semelhantes. O Major System foi projetado por Stanislaus Mink von Wennsshein do século XVII. 
O aplicativo usa um dicionário interno de mais de 50.000 palavras para criar rapidamente palavras do Major System com definições. As palavras, palavras formatadas (MaJoR) e definições podem ser copiadas simplesmente clicando nelas. Há também um popover útil que fornece uma referência de como o Major System funciona.
