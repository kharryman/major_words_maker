Este es un generador de palabras del Sistema Mayor. El Sistema Mayor es un sistema de memoria que convierte números en palabras para ayudarte a recordar números, fechas, etc. Cada uno de los 10 dígitos del sistema está representado por un conjunto de consonantes de sonido similar. El Sistema Mayor fue diseñado por Stanislaus Mink von Wennsshein en el siglo XVII.
La aplicación utiliza un diccionario interno de más de 50.000 palabras para crear rápidamente palabras del Sistema Mayor con sus definiciones. Las palabras, las palabras formateadas (MaJoR) y las definiciones se pueden copiar con un simple clic. También incluye una práctica ventana emergente que ofrece una referencia sobre el funcionamiento del Sistema Mayor.
