This is a Major System word generator. The Major System is a number to words memory system to help you remember numbers, dates, etc. Each of the 10 digits in the system is represented by a set of similar sounding consonants. The Major System was designed by Stanislaus Mink von Wennsshein of 17th Century.
The app uses an internal dictionary of over 50,000 words to quickly create major system words with definitions. The words, formatted words(MaJoR), and definitions can each be copied by simply clicking them. There is also a handy popover which provides a reference of how the Major System works.
